الحصول على المحلل للعمل في تطبيق React بسيط جدا.

## إضافة شفرة التتبع

بينما يمكنك الحصول على نسخة للنسخ واللصق مباشرة من المنصة عند إنشاء مشروع جديد ( /using-or/snippet#getting-the-snippet )، فإن هذه الشفرة قصيرة جدا وستجعل من الأمور الأكثر صعوبة في التخصيص وإضافة الإضافات الإضافية.

مع الأخذ بأنك تقوم ببناء تطبيق SPA (Single Page Application)، فإن كل ما عليك القيام به هو إضافة الشفرة التالية إلى نقطة الدخول الخاصة بك:
```jsx
import Tracker from '@openreplay/tracker';

const tracker = new Tracker({
  projectKey: "<your project key>",  
});

function YourMainComponent({props}) {
    //some code here...

    useEffect(() => {
        tracker.start();
    }, [])

    return //....
}
```
